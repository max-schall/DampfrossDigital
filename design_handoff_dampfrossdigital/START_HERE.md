# Dampfrossdigital — Design Handoff

A complete design system for **dampfrossdigital**, a digital adaptation of the *Dampfross* board game. Players build railway networks on a hex map (Network phase), then race trains across that network (Race phase).

## Start here

1. **Read `PROMPT.md`** — copy that prompt into your Claude Code session.
2. **Open `Design System.html`** in a browser to see the actual rendered designs (8 screens + components + tokens).
3. **Then read `README.md`** for the full spec.

## Local viewing

```bash
# from this folder
python3 -m http.server 8000
# then open http://localhost:8000/Design%20System.html
```

The Tweaks toggle (top of the canvas) switches between light / dark / sepia themes live.
